# Simple Todo App

This project is a simple todo app I refactored to use Object Oriented Programming.

## Functionality

This app allows the user to create todo items that can have a todo date (optional).
All input validation is handled with the FormValidator class initialized in the begining of index.js.
The creation of the todo items that are appended to the todos list in index.html are created using the ToDo class.

## Technology

This project uses JavaScript, CSS, and Html.
